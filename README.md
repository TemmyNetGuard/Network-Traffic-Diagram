# How My Data Travels to the Internet

## Overview
This diagram illustrates how data travels from a personal computer to the 
internet and back, passing through key network components including a 
Firewall and Router.

---

![How My Data Travels to the Internet](Network%20Diagram.png)

---

## How It Works

### Outgoing Traffic (Request):
1. **My Computer** sends a data request
2. **Firewall** inspects and approves the traffic
3. **Router** forwards the approved traffic online
4. **Internet** receives the request

### Incoming Traffic (Response):
1. **Internet** sends back a data response
2. **Router** receives the incoming data
3. **Firewall** inspects and approves the data
4. **My Computer** receives the approved and delivered data

---

## Key Components
- **Firewall** - Monitors and filters incoming and outgoing network traffic
- **Router** - Directs data packets between networks
- **Internet** - The global network of interconnected devices
